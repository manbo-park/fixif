# fixif

> Easy EXIF Edit

여러 사진의 EXIF를 손쉽게 수정할 수 있는 웹 앱입니다.

https://fixif.vercel.app/

---

## 기능

- **EXIF 편집** — 촬영 시간, 카메라, 렌즈, 조리개, 셔터 속도, ISO, 메모, 위치 정보 수정
- **filo 연동** — filo 앱 롤 데이터를 붙여넣어 프레임별 메타데이터 일괄 주입

---

## 사용 방법

### 1. 파일 불러오기

드롭존에 `.jpg` / `.jpeg` 파일을 드래그하거나 클릭해 선택합니다. 중복 파일명은 자동으로 건너뜁니다.

파일명에 연속 숫자가 포함된 경우(`scan_001.jpg` 등) 프레임 번호가 자동 할당됩니다.

### 2. 편집

**단일 편집**

테이블 행을 클릭하면 오른쪽 사이드 패널이 열립니다. 모든 EXIF 필드를 자유롭게 수정할 수 있으며 변경은 즉시 반영됩니다.

**일괄 편집**

체크박스로 여러 행을 선택한 뒤 툴바의 **일괄 편집** 버튼을 누릅니다. 값을 입력한 필드만 선택된 프레임 전체에 적용됩니다.

### 3. filo 데이터 붙여넣기

filo 앱에서 롤을 복사하면 클립보드에 `FILO1:...` 형식의 코드가 저장됩니다.

fixif 화면에서 아무 입력 필드 밖에 **Ctrl+V** (또는 Cmd+V)를 누르거나, 툴바의 **filo 데이터 붙여넣기** 버튼을 사용합니다. 프레임 번호 기준으로 자동 매칭 결과를 미리보기한 뒤 **적용**을 누르면 한 번에 주입됩니다.

### 4. 이미지 미리보기 및 회전

썸네일을 클릭하면 미리보기 모달이 열립니다.

| 동작                | 단축키        |
| ------------------- | ------------- |
| 이전 / 다음 이미지  | ← → 화살표 키 |
| 왼쪽으로 90° 회전   | Q             |
| 오른쪽으로 90° 회전 | E             |
| 회전 초기화         | R             |
| 모달 닫기           | Esc           |

### 5. 내보내기

툴바 오른쪽의 **새 파일로 내보내기** 버튼을 누릅니다.

- 선택된 행이 있으면 선택된 파일만 내보냅니다.
- 선택이 없으면 전체 파일을 내보냅니다.

입력 오류가 있는 행이 하나라도 있으면 내보내기가 비활성화됩니다.

## 단축키

| 단축키                     | 동작                                    |
| -------------------------- | --------------------------------------- |
| Ctrl+Z / Cmd+Z             | 실행 취소 (Undo)                        |
| Ctrl+Shift+Z / Cmd+Shift+Z | 다시 실행 (Redo)                        |
| Ctrl+V / Cmd+V             | Filo 데이터 붙여넣기 (입력 필드 밖에서) |
